## Description

FlipFit recruitment task.

## Prerequisites
Node 12+

## Installation

```bash
$ npm i
```

## Setup environment variables

```bash
$ CP .env.dist .env
```
Provide or override default values as needed.

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Documentation
API is documented using swagger. To open the docs go to http://localhost:3000/api.

## Tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
